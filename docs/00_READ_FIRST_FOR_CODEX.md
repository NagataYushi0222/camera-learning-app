# Read First for Codex

このプロジェクトは、カメラと目の仕組みを学ぶブラウザアプリです。

目的は、教科書的な光線図だけでは分かりにくい以下の本質を、3Dビューとスクリーン表示を通して理解できるようにすることです。

- 物が見えるとは何か
- なぜ光が届くだけでは像にならないのか
- なぜレンズやピンホールが必要なのか
- レンズは何をしているのか
- ピントが合うとは何か
- 目とカメラはどう同じなのか

画面構成の基本は次の通りです。

- 左側：自由に移動できる3Dビュー
- 右側：3D空間内のスクリーン / センサーに実際に写っている像
- 下部：ノベルゲーム風の解説ボックス
- 右側スクリーンをクリックすると、その1ピクセルに届く光の経路が左側3Dビューで可視化される

最重要機能は「センサー1ピクセル解析モード」です。

この機能により、ユーザーはスクリーン上のある点をクリックして、
そのピクセルがどこから来た光で構成されているのかを確認できます。
