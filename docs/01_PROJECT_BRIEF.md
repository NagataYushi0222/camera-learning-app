# Project Brief

## App concept

「見えるとは何か？」を出発点にして、カメラと目の仕組みを体験的に学ぶWebアプリ。

## Core message

見えるとは、単に光が届くことではない。

物体の各点から来た光が、スクリーンや網膜上の各点に対応づけられることで、像として見える。

## Main learning loop

1. 解説を読む
2. 3Dビューで位置関係を見る
3. 右側のスクリーン上の像を見る
4. スクリーン上の1点をクリックする
5. そのピクセルに届く光の経路を3Dビューで確認する
6. なぜそのように見えるのかを理解する

## Design direction

ノベルゲームのように、学習者がガイドに案内されながら進む。

ただし、説明とシミュレーションを完全に分離しすぎない。
基本は、説明・3Dビュー・スクリーン結果を同じ学習体験の中で接続する。

場面によっては、説明が中心のストーリーパートを挟む。
その後、3Dで確かめるパートに進む。
