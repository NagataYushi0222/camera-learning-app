# Sensor One-Pixel Analysis Mode

## Overview

右側のスクリーン / センサー上の任意の1点をクリックすると、
そのピクセルに届いている光の出どころを左側の3Dビューで表示する。

この機能は、このアプリの最大の特徴である。

## Educational purpose

通常の教材では、スクリーンに像ができる理由を外側から説明するだけになりやすい。

このアプリでは、スクリーン上の1ピクセルを入口にして、
そのピクセルがなぜその色・明るさになったのかを逆向きにたどる。

## Modes

### No lens mode

レンズもピンホールもない。

スクリーン上の1点をクリックすると、
物体全体や背景など、さまざまな場所から光が届いている様子を表示する。

学習者に伝えること：

- 光が届くだけでは像にならない
- 1ピクセルに多くの場所からの光が混ざると位置情報が失われる
- その結果、スクリーン上には形が現れない

### Pinhole mode

ピンホールを通れる光だけがスクリーンに届く。

スクリーン上の1点をクリックすると、
その点に届く光がピンホールによって限定されていることを示す。

学習者に伝えること：

- ピンホールは光を曲げない
- 通れる光を制限することで位置対応を作る
- 像は暗いが形が現れる

### Lens in-focus mode

レンズがあり、ピントが合っている。

スクリーン上の1点をクリックすると、
物体上の対応する1点から広がった光が、
レンズによって曲げられ、
そのスクリーン上の1点に集まる様子を表示する。

学習者に伝えること：

- レンズは光をただ集めるのではない
- 同じ物体点から出た光を同じセンサー位置に集める
- これにより明るくシャープな像ができる

### Out-of-focus mode

スクリーン位置がずれている。

スクリーン上の1点をクリックすると、
本来1点に集まるはずの光が広がっている様子を表示する。

学習者に伝えること：

- ピントが外れるとは、物体の1点がセンサー上で広がること
- ボケは、点が点として写らない現象である
