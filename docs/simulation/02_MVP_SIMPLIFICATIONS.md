# MVP Simplified Optics Model

初期MVPでは、見た目と学習フローを優先するため、光学計算は教育用の簡易モデルとして実装する。

- 光線は、波の進行方向を説明する補助線として表示する。
- レンズなしでは、選択ピクセルに複数の物体点から光が届く状態を、複数の直線光線で表す。
- ピンホールでは、物体側の点から穴を通ってセンサーへ届く折れ線として表す。光は曲げず、穴で経路を制限する。
- レンズありでは、同じ物体点から出た複数の光線がレンズ面を経由して選択ピクセルへ集まるように表す。
- ピントずれでは、対応点の周辺から来た光が選択ピクセル付近で広がる状態を、簡易的なボケ表現として扱う。

この段階では、厳密な波動光学、干渉、回折、レンズ収差、実焦点距離計算は行わない。
後続フェーズで、各モードの教育目的に合わせて計算精度と表示を段階的に高める。
